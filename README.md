# Aurionth

> *Webová stránka o mém fiktivním světě.* — Aurittinth Kîllis No'Uu

**Aurionth** je rozsáhlý fantasy světotvorný projekt zahrnující dvanáct tematických atlasů, konstruovaný jazyk Novorodštinu, pantheon božstev, bestiář a genealogie postav. Tento repozitář obsahuje zdrojový kód webu, který slouží jako interaktivní encyklopedie světa Aurionth.

🌐 **[aurittinth.github.io/Aurionth](https://aurittinth.github.io/Aurionth/)** (Možná někdy jako `aurionth.ath`)

---

## Obsah projektu

- **Atlasy** — dvanáct tematických celků pokrývajících geopolitiku, kultury, náboženství a historii světa
- **Novorodština** — konstruovaný jazyk s vlastní gramatikou, deklinacemi a slovníkem
- **Pantheon & náboženství** — propracované věrouky včetně Xērroroi, archetypálních kosmických bytostí
- **Bestiář** — katalog tvorů světa Aurionth
- **Genealogie** — rodokmeny klíčových postav a dynastií
